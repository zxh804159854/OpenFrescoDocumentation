# Tcl 命令语言简介

OpenFresco 命令基于 Tcl 脚本语言实现。用户通过执行命令创建混合模拟所需的各类对象，包括试验单元、试验场地、试验装置与试验控制器等，本文档将对这些对象逐一说明。Tcl 语言提供变量、数学表达式计算、控制结构等实用编程工具，支持用户编写脚本以定义混合模拟流程。

Tcl 是一种基于字符串的脚本语言，支持以下功能：

- 变量与变量替换

- 数学表达式计算

- 基本控制结构（if、while、for、foreach）

- 过程（子程序）定义

- 文件操作

Tcl/Tk 入门手册：<http://www.tcl.tk/man/tcl8.5/TclCmd/contents.htm>
