# 版本更新说明

## 2.6 版本更新说明

**试验单元建模**

openFresco 试验单元嵌入有限元软件中，用于表征结构中开展物理试验的部分，并为有限元分析程序提供必要的交互接口。

在2.5版本既有试验单元基础上，2.6版本新增以下功能：

1.  对**双节点连接单元（twoNodeLink）**进行修改：当单元长度大于0时，可考虑剪力与弯矩的耦合作用；若单元非零长度，用户可自定义局部x、y轴P-Delta效应弯矩在i节点弯矩、j节点弯矩与剪力偶之间的分配比例，三者比例之和恒为1。

2.  新增**共旋桁架试验单元（corotTruss）**，可用于大位移工况下的桁架单元试验。该单元采用共旋列式，坐标系随单元同步转动，可精确实现局部与整体坐标系间的几何变换。

**试验装置表征**

OpenFresco 中的试验装置对象是实验室真实试验装置的软件抽象，负责依据加载与测试系统的几何关系和运动学特性，在试验单元自由度与作动器自由度之间完成响应量的双向转换。

2.6版本对试验装置模块进行多项改进与新增：

1.  新增**三维四作动器试验装置（FourActuators3d）**，用于三维柱体试验。该装置通过四个作动器控制框架试件的两个平动与两个转动自由度，不控制轴向与扭转自由度，但可考虑恒定轴向荷载（通过分配梁与两根预应力拉杆施加）对单元抗力的影响。该装置所有转换均采用非线性变换，可适配大位移工况。

2.  试验装置中的响应修正系数可分别作用于**试算响应量**（转换为控制信号前）与**输出响应量**（由采集信号转换后）。该功能可对试算、输出、控制与采集响应量分别施加修正系数，满足相似律缩放的精确计算需求。

**与控制器及数据采集系统交互**

OpenFresco 提供丰富的函数库，用于与实验室控制及数据采集系统通信，可在作动器坐标系下发送控制指令，并从采集系统获取实测数据。

2.6版本在该模块实现重大功能升级：

1.  修复了与3.1以上版本 xPC-Target 兼容时的死锁问题，xPCtarget 试验控制模块可完美适配当前 xPC-Target 4.1版本（Matlab R2009a）。

2.  试验控制对象可搭载信号滤波对象，对控制与采集响应量进行滤波，并可模拟超调、欠调、超前、滞后及随机噪声等试验误差。

**试件模拟（含控制器与数据采集系统）**

为在正式试验前校验混合模型并预估结构响应，可通过 OpenFresco 模拟试验控制与采集对象实现试件虚拟模拟。2.6版本在既有 SimUniaxialMaterial、SimDomain 试验控制类基础上，新增 **SimFEAdapter 类**。

该对象支持用户在任意选定的有限元软件中模拟物理子结构与试件。目前已为 OpenSees、LS-DYNA、Abaqus 开发适配单元，作为主程序与子结构模拟有限元软件的交互接口。该适配单元模式无需通过文件系统在主、从有限元软件间交换数据，也无需反复启停程序，可实现混合模拟的连续并行运行。

**多数值计算驱动支持**

为适配多款有限元软件，OpenFresco 采用三层软件架构，并在有限元软件中集成通用客户端单元。只要有限元软件提供应用程序编程接口（API）支持用户自定义单元，即可嵌入该通用客户端单元；且同一有限元软件仅需实现一次通用客户端单元，即可作为混合模拟的数值计算驱动。

目前已为 OpenSees、Matlab、LS-DYNA、UI-SimCor、Abaqus、Simulink 完成通用客户端单元开发，其中 **Abaqus 与 Simulink 接口为2.6版本新增**。

**试验数据记录**

记录器类可对试验人员关注的所有试验对象（单元、场地、装置、控制器等）的响应量进行记录，支持多种数据格式：无表头ASCII格式、无表头ASCII逗号分隔格式、含XML元数据的ASCII格式及二进制格式。此外，响应数据可写入 SQL、Berkeley DB 等数据库。

**OpenFresco 软件安装**

随着 OpenFresco 支持的控制器与数据采集系统接口库快速扩展，所需动态链接库数量也同步增长。与此同时，用户与开发者的安装流程需尽可能简洁灵活。

2.6版本沿用2.5版本采用的 NSIS（Nullsoft 脚本安装系统）开源专业安装程序，为 Windows 平台提供面向普通用户与开发者的安装选项。

**OpenFresco 文档**

2.6版本新增多篇核心文档，以公开评审形式发布，用于说明 OpenFresco 各模块功能，经短期公开评审后将正式定稿。新增文档内容包括：

1.  软件安装与入门指南

2.  OpenFresco 专用 TCL 脚本命令

3.  混合模拟算例

4.  有限元软件（OpenSees、Abaqus、LS-DYNA、Matlab）混合试验适配方法

5.  安全通信通道所需证书生成方法
